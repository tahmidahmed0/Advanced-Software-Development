# Agile Methodology Report Template

## 1. Introduction
This project used an Agile approach because the requirements were broad and the system needed iterative design, development, testing, and review.

## 2. Why Agile Was Chosen
- Supports incremental delivery
- Helps divide work across group members
- Allows changes after feedback
- Matches weekly lab-based development
