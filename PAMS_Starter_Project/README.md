# Paragon Apartment Management System (PAMS)

Desktop application starter project for the UFCF8S-30-2 Advanced Software Development portfolio.

## Stack
- Python 3.11+
- Tkinter GUI
- SQLite database
- Object-oriented structure
- unittest for automated tests

## Features included
- Role-based login
- Tenant management
- Apartment management
- Lease and payment logic
- Maintenance request tracking
- Basic reports
- Input validation
- Password hashing
- Sample mock data

## Run
```bash
python main.py
```

Default demo users:
- admin / Admin123!
- frontdesk / Front123!
- finance / Finance123!
- maintenance / Maint123!
- manager / Manager123!
